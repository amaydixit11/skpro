# ESoC 2026 — Program Details

## What Is ESoC?

**European Summer of Code (ESoC) 2026** is a European initiative that funds stipends for contributors new to open source worldwide, matching them with open source projects and applied AI projects across Europe.

- **Inspired by:** Google Summer of Code (GSoC)
- **Hubs:** Decentralized across Europe
  - 🇫🇷 probabl (scikit-learn company)
  - 🇩🇪 German Center for Open Source AI (GC.OS) ← **your hub**
  - 🇮🇹 Fondazione Bruno Kessler
- **Format:** Virtual via GitHub & Discord, with optional hybrid/in-person

## Expectations

### Time Commitment
- **12 calendar weeks** of full-time dedication
- **5 working days of pause** (paid)
- Flexible scheduling, but 3 months continuous availability expected

### What You'll Do
- Weekly 1-on-1 mentoring meetings
- Daily contact with mentors via Discord
- Regular code/doc/blog post submissions
- Peer-review of another intern's work (mandatory)
- Active community engagement

### What You Get
- Flat stipend of €4,800 (not formal employment)
- Rapid feedback (typically within 2 business days)
- Portfolio building
- Integration into the sktime/skpro developer community
- Potential for long-term open-source contribution/leadership

### Code of Conduct
- Must follow the project Code of Conduct
- Daily mentor communication
- Active participation in community/mentee activities

## Interview Process

1. **Code Presentation (5-10 min):** Walk through Python code you wrote (screen-share or repo link)
2. **Q&A:** Discussion of motivation, previous experience, project suitability, technical questions on data science & Python
3. **Your Questions:** Opportunity to ask the interviewers questions
4. **Outcome:** Conditional acceptance (shortlisted for ESoC reconciliation) or rejection — communicated within ~1 week
